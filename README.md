## Как реализовано

### 1) Фон
В `body` добавлен градиент + фоновое изображение:

```css
body {
  background:
    linear-gradient(180deg, rgba(2, 7, 26, 0.8) 0%, rgba(3, 9, 30, 0.92) 100%),
    url("https://museumgagarin.ru/images/1home-03-slide-02.jpg");
  background-size: cover;
  background-attachment: fixed;
}
```

### 2) Лого/бренд
В разметке добавлен хедер с названием проекта:

```html
<header class="gagarin-topbar">
  <div class="gagarin-brand">
    <p class="gagarin-brand-caption">ФГБУК МУЗЕЙ-ЗАПОВЕДНИК</p>
    <p class="gagarin-brand-name">Ю. А. ГАГАРИНА</p>
  </div>
</header>
```

### 3) Новый цвет кнопок
Обычные и операционные кнопки окрашены разными цветами:

```css
.my-btn {
  background: rgba(26, 56, 112, 0.85);
}

.my-btn.primary {
  background: rgba(255, 104, 37, 0.93);
}
```
